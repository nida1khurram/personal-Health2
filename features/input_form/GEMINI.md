# Day 2: Health Record Input Form

## Goal
Build an input system for daily health data entry.

## Learning Focus
- Streamlit widgets
- Data validation
- CSV file handling

## Features
1. Input: Date, Blood Pressure, Sugar Level, Pulse Rate, Notes  
2. Save to CSV (`data/health_records.csv`)  
3. Validation for numeric fields  
4. Confirmation message on save  

## Streamlit Components
- st.date_input()
- st.number_input()
- st.text_area()
- st.button()

## Success Criteria
✅ Able to add new record  
✅ Data saved correctly  
✅ Confirmation displayed  
✅ Simple & user-friendly UI
